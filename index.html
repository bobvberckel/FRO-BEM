Wie ben ik?

Mijn naam is Bob van Berckel, ik ben 17 jaar oud en woon in Hilversum. Ik volg de opleiding Media Development (het ontwikkelen van websites en webapplicaties) aan het Media College in Amsterdam. Vanaf mijn 13e jaar ben ik al actief bezig met bijbaantjes. Af en toe doe ik ook al wat kleine online projecten in opdracht.

Wat zijn mijn vaardigheden?

<html>

Ik heb sterke kennis als het aankomt op HTML, het is ook niet heel erg moeilijk. Ik kan er goed mee overweg en heb geen problemen tijdens het schrijven van HTML.

.css { }

Bij HTML hoort natuurlijk ook CSS, ik kan juist met CSS overweg gaan. Maar ben niet in staat om daadwerkelijk iets heel moois er van te maken. Daarnaast schrijf ik ontzettend veel code als het aankomt bij CSS, wat soms dubbel op is.

console.log(`Javascript`);

Bij het maken van een website is soms Javascript nodig om met de gebruikerinteractie mogelijk te maken. Ik ben diep in Javascript gedoken en schrijf nette, leesbare code die in meerdere manieren gebruikt wordt. (Helpdesk generator, Discord Bot). Soms gaat het echter wel iets té ver boven mijn hoofd waardoor ik het niet altijd even snel begrijp.

echo "php";

Bij het maken van iets complexere websites is het nodig om PHP te schrijven, ik heb basis kennis van PHP en kan dit op de juiste manier gebruiken bij projecten. Ik ben nu bezig met het leren van OOP PHP code schrijven, waardoor ik mijn eigen geschreven code opnieuw kan gebruiken.

print("python")

Momenteel ben ik bezig met het leren van Python, dit wil ik uiteindelijk ook gebruiken in websites (Django). Ik ken de basis, maar heb nog een lange weg te gaan voordat ik kan zeggen dat ik het echt onder de knie heb.
